Instructions

There are two modes to this application, Display Mode (represented by an eye icon) and Edit Mode (represented by a pencil icon). The former allows you to view your memories on the map. The latter allows you to edit and add new pinpoints to the map. Proceed with Edit Mode to create your first marker.

Display Mode

  • Click a marker to preview a memory.
  • Click the preview to open up the memory info panel.
  • Click the marker again to hide the preview.

Edit Mode

  • Click a marker to open up the memory edit panel.
  • Fill out the memory form and hit submit.
  • Click the marker in Display Mode to view the memory.
